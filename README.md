# WebGitNet

A Github (Bitbucket, Gitlab, Gitbucket, Gitorious) alternative that you can self
host on your Windows server.

## Installation

Checkout out the [Getting Started](https://github.com/otac0n/WebGitNet/wiki/Getting-Started)
